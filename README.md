# 🛒 Grocery Store Management System

A simple full-stack Grocery Store Management System built using Python for backend processing and HTML, CSS, and JavaScript for the frontend interface.

This project demonstrates CRUD operations, database connection, and basic full-stack integration.

---

## 🚀 Features

- Add new products
- Manage product list
- Create customer orders
- Database connectivity
- Interactive UI

---

## 🛠 Technologies Used

### Backend
- Python
- Flask (if applicable)
- MySQL

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap
